# EcoHealth AI Agent
Agente inteligente para consultar la calidad del aire y recibir consejos de salud según la ciudad proporcionada (orientado a asmáticos).

## Cómo ejecutar
1. Crea tu entorno: `python -m venv venv`
2. Instala todo: `pip install -r requirements.txt`
3. Configura tus claves en el archivo `.env`.
4. Lanza el servidor: `uvicorn web_app:app --reload`
